# Student Depression Prediction Using Machine Learning

## Description
Developed a machine learning classification project to predict student depression based on academic, behavioral, and personal factors.

## Models Used
- Decision Tree
- Naive Bayes
- K-Nearest Neighbors (KNN)
- Random Forest

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Machine Learning
- Data Analysis

## Workflow
- Data preprocessing
- Feature selection
- Categorical encoding
- Model training
- Model evaluation
- Performance comparison
